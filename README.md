# Bayard Vacations AI Chatbot

This project is a simple AI-powered chatbot designed to assist customers with basic travel-related queries.

## Features
- Answers questions about travel packages
- Provides popular destinations
- Shares pricing information
- Explains booking process

## Technologies Used
- Python
- Streamlit
- Rule-based NLP logic

## How to Run
1. Install dependencies:
   pip install streamlit
2. Run the app:
   streamlit run app.py

## Limitations
- Rule-based responses
- Cannot handle complex or dynamic queries

## Future Improvements
- NLP models like BERT
- Database integration
